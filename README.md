# 📱 Aplicativo de Gerenciamento de Pessoas 

Esta é uma aplicação mobile desenvolvida para executar as operações de CRUD (Create, Read, Update e Delete) em um banco de dados armanezado em JSON. As informações arquivadas são exibidas em formatos de card no Aplicativo que estará rodando pelo celular.
